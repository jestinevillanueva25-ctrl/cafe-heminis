Phone scanners cannot access local server URLs (localhost/file) directly. Connect both devices to the same Wi-Fi network and input your computer's local IP address above to test.

How do I find my local IP address?
  1. Open Terminal on your Mac.
  2. Run ipconfig getifaddr en0 (or check System Settings > Wi-Fi > Details).
  3. Make sure your phone is on the same Wi-Fi network.
  4. Enter http://<your-ip>:8080 (if running a local server) in the input above and click Update.
